- Installation -

SFTP add-on adds SSH File Transfer Protocol support to JFileUpload and others add-ons such as JImageUpload, JImageFilter, JBatchUpload and JDiskExplorer. It allows having secure file transfer over SSH.

Instructions

SFTP is a JFileUpload add-on so you do need JFileUpload installed before installing it. Here are instructions :

  1. Copy dist/lib/sftpimpl.jar and jsch.jar under lib/ folder (where others JFileUpload JAR files are stored).
  2. Copy applet_sftp.html and applet_sftp.js files under the same folder as JFileUpload HTML/JavaScript files.
  3. Modify dist/applet_sftp.js to setup basic parameters and open the HTML file in your browser to try it.
Parameters

Parameters have to be setup in applet_sftp.js JavaScript file. You can select SFTP url (including hostname and port), SSH username and password. You benefit from all JFileUpload parameters such as resume, zip-on-fly, automatic retry on transfer failure and more. See SFTP related parameters here.
